C语言基础知识

掌握C语言编程的核心概念和基础语法

变量和数据类型

  • 整型（int）、浮点型（float, double）
  • 字符型（char）
  • 变量声明和初始化
  • 常量定义（const）

示例代码

int age = 25;
float height = 175.5f;
char grade = 'A';
const double PI = 3.14159;

运算符和表达式

  • 算术运算符（+, -, *, /, %）
  • 关系运算符（>, <, ==, !=）
  • 逻辑运算符（&&, ||, !）
  • 自增自减（++, --）

示例代码

int a = 10, b = 5;
int sum = a + b;    // 15
int product = a * b;  // 50
bool isGreater = a > b;  // true

控制结构

  • if-else 条件语句
  • switch-case 选择语句
  • for 循环
  • while 和 do-while 循环

示例代码

// if-else 示例
if (score >= 60) {
    printf("及格\n");
} else {
    printf("不及格\n");
}

// for 循环示例
for (int i = 0; i < 5; i++) {
    printf("%d ", i);
}

函数

  • 函数定义和声明
  • 参数传递
  • 返回值
  • 函数原型

示例代码

int add(int a, int b) {
    return a + b;
}

void printMessage(char* msg) {
    printf("%s\n", msg);
}
