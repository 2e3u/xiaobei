动态内存管理

  • malloc() 和 free()
  • 动态数组
  • 内存泄漏防范
  • calloc() 和 realloc()

示例代码

// 动态内存分配
int* dynamicArray = (int*)malloc(5 * sizeof(int));
if (dynamicArray != NULL) {
    // 使用内存
    for (int i = 0; i < 5; i++) {
        dynamicArray[i] = i + 1;
    }
    // 释放内存
    free(dynamicArray);
}
