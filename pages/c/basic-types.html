C语言基础类型

掌握C语言的基本数据类型和变量声明

整数类型

C语言提供了多种整数类型，用于表示不同范围的整数值。

示例代码

// 整数类型声明
int age = 25;              // 标准整型
short small = 123;         // 短整型
long big = 1234567890L;    // 长整型
unsigned int positive = 100;// 无符号整型

// 整数类型的范围
printf("int 类型范围：%d 到 %d\n", 
    INT_MIN, INT_MAX);

要点说明

  • int 是最常用的整数类型
  • 使用 unsigned 表示无符号数
  • long 用于存储较大的整数
  • 不同类型占用不同的内存空间

浮点类型

浮点类型用于表示小数，C语言提供了单精度和双精度两种浮点类型。

示例代码

// 浮点类型声明
float price = 19.99f;       // 单精度浮点数
double pi = 3.14159265359;  // 双精度浮点数

// 科学计数法
float scientific = 1.23e-4f;// 0.000123

字符类型

示例代码

// 字符类型声明
char grade = 'A';           // 字符
char newline = '\n';        // 转义字符

// ASCII值
int ascii_value = grade;    // 65
