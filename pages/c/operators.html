运算符和表达式

掌握C语言中各种运算符的使用和表达式的计算

算术运算符

用于执行基本的数学运算。

示例代码

int a = 10, b = 3;

// 基本运算
int sum = a + b;      // 加法: 13
int diff = a - b;     // 减法: 7
int product = a * b;  // 乘法: 30
int quotient = a / b; // 除法: 3
int remainder = a % b;// 取余: 1

// 自增自减
int x = 5;
x++;                  // 后自增: x = 6
++x;                  // 前自增: x = 7
x--;                  // 后自减: x = 6
--x;                  // 前自减: x = 5

注意事项

  • 整数除法会舍弃小数部分
  • 取余运算只适用于整数

关系运算符

示例代码

int x = 5, y = 8;

// 比较运算
bool isEqual = (x == y);    // 等于: false
bool notEqual = (x != y);   // 不等于: true
bool isLess = (x < y);      // 小于: true
bool isGreater = (x > y);   // 大于: false
bool lessEqual = (x <= y);  // 小于等于: true
bool greaterEqual = (x >= y);// 大于等于: false

逻辑运算符

示例代码

int age = 25;
bool hasID = true;

// 逻辑运算
bool canDrink = (age >= 18) && hasID;  // 逻辑与
bool isChild = (age < 12) || (age > 65);// 逻辑或
bool isNotAdult = !(age >= 18);        // 逻辑非

位运算符

示例代码

unsigned int a = 60;  // 二进制: 0011 1100
unsigned int b = 13;  // 二进制: 0000 1101

// 位运算
int c = a & b;       // 按位与: 0000 1100
int d = a | b;       // 按位或: 0011 1101
int e = a ^ b;       // 按位异或: 0011 0001
int f = ~a;          // 按位取反: 1100 0011
int g = a << 2;      // 左移: 1111 0000
int h = a >> 2;      // 右移: 0000 1111
